# Super Admin Frontend

This project is a modern React application designed for managing various administrative tasks. It utilizes Tailwind CSS for styling, providing a responsive and user-friendly interface.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd super-admin-frontend
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

The application will be available at `http://localhost:3000`.

## Usage

This application provides a dashboard for managing admins, campuses, groups, programs, and students. Each section includes forms for creating and editing entries, as well as management views for listing and interacting with existing data.

## Components

- **SuperPanel**: Main dashboard for navigation.
- **Admin Management**: Create, edit, and manage admin users.
- **Campus Management**: Create, edit, and manage campuses.
- **Group Management**: Create, edit, and manage groups.
- **Program Management**: Create, edit, and manage programs.
- **Student Management**: Create, edit, and manage students.
- **Login**: Admin login interface.
